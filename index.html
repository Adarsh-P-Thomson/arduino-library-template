Quick Start

  1. Click "Use this template" button on GitHub to create your repository
  2. Clone your new repository to your computer
  3. Run the setup script: ./setup-library.ps1 (Windows) or ./setup-library.sh (Linux/Mac)
  4. Implement your library code in the src/ directory
  5. Update examples in examples/ to demonstrate your library
  6. Push to GitHub - CI runs automatically to validate your code
  7. Create a release tag and submit to Arduino Library Manager
